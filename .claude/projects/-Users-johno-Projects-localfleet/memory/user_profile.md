Johno is building LocalFleet — a multi-domain LLM command & control system for autonomous fleet simulation (surface vessels + drones). Runs entirely local on Mac Studio M3 Ultra with 256GB RAM, air-gapped, using Ollama (Qwen 72B) for local LLM inference. Forked from CORALL for vessel dynamics/COLREGS/risk.

Prefers strict rules: schemas are source of truth, one file per session, test before moving on, never refactor working code, build bottom-up.
