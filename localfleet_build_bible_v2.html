Technical Risks & Mitigations

Risk: CORALL code issues on Apple Silicon

Mitigation: CORALL is pure Python (numpy, scipy, matplotlib). All have native ARM wheels. If matplotlib GUI has issues, run headless — your React dashboard replaces it anyway.

Risk: Local LLM too slow or bad quality

Mitigation: Your M3 Ultra runs 70B at ~10-20 tok/s. Most commands need 50-100 tokens = 3-5 seconds worst case. If too slow, drop to 32B. Also: cache common decisions.

Risk: Multi-domain parsing is hard for the LLM

Mitigation: Your system prompt explicitly lists available assets by domain. "Available surface assets: alpha, bravo, charlie. Available air assets: eagle-1." The LLM just needs to route commands to the right asset IDs. Structured output mode forces valid JSON.

Risk: Drone integration takes too long

Mitigation: The drone model is trivially simple — 50 lines of Python max. It's a dict with x, y, altitude, heading, speed that gets updated each tick. If it takes more than 2 days, you've over-engineered it. STOP and simplify.

Risk: You get stuck and lose momentum
